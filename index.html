Kde suggestions

A view from the average user

1st lets layout the useage of kde then i will go into the positive and negitive aspects that people said about KDE.

I had people install manjaro KDE on laptops with the only intervention being to add a swap space. The instalation went smooth with no hickups or questions about what was going on.

When they finished installing i only installed 3 programsfor them to use. KDE discover, appimagepool, and sshd (this is only for myself).

Finding and using programs using these 2 "stores" was easy there was no complants only positive feed back from this.

General usage of KDE plasma has both positve and negitive feedback

Positive feedback

           -very customizable
           -easy to learn from windows
           -Fast
           -Looks pretty
           -KDE connect
           -virtural desktops

Negitive

           -bluetooth earbuds did not work
           -cant move system trey icons
           -cant use a network drive in any program exept dolphin (I setup fstab but they wanted an experence like windows network drives.)

This is the feedback i got form the people hepling me to see if a average person can actually just pick up and use linux.

           
           Endnote
           the reason KDE plasma was used is because it looked and behaved the most like windows. I originally wanted to test diffrent Desktop enviroments but KDE plasma was by far the most liked DE (out of Gnome,Cinnamon,XFCE and MATE.)

Pwnagotchi

A digital pet for cyber security, Jun 13, 2021

Pwnagotchi is an A2C-based “AI” powered by bettercap and running on a Raspberry Pi Zero W that learns from its surrounding WiFi environment in order to maximize the crackable WPA key material it captures

Instead of merely playing Super Mario or Atari games like most reinforcement learning based “AI” (yawn), Pwnagotchi tunes its own parameters over time to get better at pwning WiFi things in the real world environments you expose it to.
